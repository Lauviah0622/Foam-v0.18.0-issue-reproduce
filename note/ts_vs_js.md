# ts_vs_js


|                  | JS                                         | TS                                  |
|------------------|--------------------------------------------|-------------------------------------|
| 變數與型別的綁定 | 動態                                       | 靜態語言                            |
| 型別自動轉換     | 是                                         | 否                                  |
| 型別檢查         | 執行時期 runtime                           | 編譯時期 compile                    |
| 錯誤             | runtime，不過像是語法錯誤這種就是在 compile | 型別錯誤這種就會在 runtime 裡面出現 |



因為 JS 在執行的時候才會去推斷型別，所以在 compile 時期不會知道你的型別有什麼錯誤
